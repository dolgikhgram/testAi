## Запуск

Установи зависимости и запусти dev-сервер:

```bash
npm install
npm run dev
```

Открой [http://localhost:3000](http://localhost:3000) в браузере.

### Другие команды

- `npm run build` - сборка для production
- `npm start` - запуск production сервера
- `npm test` - запуск тестов
- `npm run lint` - проверка кода

## Что есть

**Основные возможности:**
- Создание и управление несколькими диалогами
- Поиск по диалогам
- Переименование и удаление диалогов
- Стриминг ответов 
- Автосохранение в localStorage
- Адаптивный дизайн
- Поддержка темной темы

**UI компоненты:**
- Button, Input, Modal, CodeBlock
- Компоненты для сообщений и списка диалогов
- Все компоненты в отдельных папках с CSS Modules


## Технологии

Next.js 16, React 19, TypeScript, CSS Modules, Jest, clsx, framer-motion

## Тестирование

В проекте есть юнит-тесты для основных компонентов и функций:
- Тесты для функций работы с localStorage 
- Тесты для UI компонентов (Button, Input)
- Тесты для компонента Message
